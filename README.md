# 6 to 9 Dental Texas - Hero Section Redesign

A modern, high-converting hero section redesign for **6 to 9 Dental Texas**, built as a cold outreach demo. 

## 🚀 Overview

This project is a targeted frontend redesign aimed at improving the user experience and conversion rate of the original 6 to 9 Dental website. The new hero section addresses previous design inconsistencies (such as outdated textures, clashing colors, and lack of visual hierarchy) by implementing a clean, accessible, and trust-building layout.

### ✨ Key Features
- **Modern UI/UX**: Calming sage green and warm off-white color palette to convey cleanliness and trust.
- **Clear Call-to-Actions**: Prominent, highly visible CTA buttons strategically placed to drive appointment bookings.
- **Trust Indicators**: Clean trust badges highlighting the clinic's stellar ratings, VA Vouchers acceptance, and PPO coverage using crisp SVG icons.
- **Fully Responsive**: Mobile-first design that looks beautifully structured across all device sizes.
- **Optimized Assets**: Utilizes Next.js Image optimization for fast, performant loading.

## 🛠 Tech Stack

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Typography**: Inter (Google Fonts)
- **Icons**: `country-flag-icons`

## 💻 Running Locally

To run the development server locally:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🎨 Design Philosophy
The design philosophy centers around a "warm, family-friendly, modern-but-approachable" aesthetic. Hard lines and harsh colors have been replaced with rounded elements, generous whitespace, and a unified typography system to establish immediate credibility.
